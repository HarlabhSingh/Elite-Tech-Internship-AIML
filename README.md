# Elite Tech Internship - AI/ML Track

This repository contains the core Machine Learning projects and tasks completed during my virtual internship at Elite Tech Intern. The projects demonstrate fundamental concepts in supervised and unsupervised machine learning using Python.

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Libraries:** Scikit-Learn, NumPy, Pandas, Matplotlib
* **Environment:** Google Colab

---

## 📋 Completed Tasks

### Task 1: Housing Price Prediction (Multiple Linear Regression)
* **Objective:** Predict real estate prices based on square footage, number of bedrooms, and number of bathrooms.
* **Concepts Covered:** Supervised learning, multiple features, baseline intercepts, and evaluating model coefficients.

### Task 2: Customer Segmentation (K-Means Clustering)
* **Objective:** Group retail store customers into distinct profiles based on their purchase history metrics (RFM: Recency, Frequency, Monetary value).
* **Concepts Covered:** Unsupervised learning, Euclidean distance, feature scaling via StandardScaler, and the Elbow Method for finding optimal clusters.

### Task 3: Image Classification (Support Vector Machine)
* **Objective:** Implement a Support Vector Machine (SVM) classifier to categorize and process image data.
* **Concepts Covered:** Image flattening, feature arrays, hyperparameter tuning, and classification accuracy scoring.
